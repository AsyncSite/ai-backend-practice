# [Session 10] AI 활용 프롬프트 -- 아키텍처

## 1단계: 이해

```
헥사고날 아키텍처와 계층형 아키텍처의 차이를 설명해줘.
각각의 장단점과 어떤 규모의 프로젝트에 적합한지 알려줘.
```

## 2단계: 적용

```
Spring Boot로 계층형 아키텍처 프로젝트의 패키지 구조를 만들어줘.
모듈: 회원, 상품, 주문, 결제
각 모듈에 Entity, Repository, Service, Controller를 포함해줘.
```

## 3단계: 검증

```
이 구조에서 Service가 직접 다른 모듈의 Repository를 호출하면 어떤 문제가 발생해?
모듈 간 결합도를 낮추는 방법을 알려줘.
```
